# Codex tickets — Roadmap Task-DSL + Action Log + UUID

Deze map bevat tickets/prompts om met Codex (VS Code) stapsgewijs de nieuwe architectuur te implementeren:

- Startstructuur -> takenformulier -> export
- Task DSL (expliciete taken)
- Action log (events)
- UUID tracking voor nodes
- Evaluator die taken valideert op basis van log + state

Werkwijze:

1. Open één ticketbestand.
2. Plak de "Codex prompt" in Codex.
3. Laat Codex de wijzigingen uitvoeren.
4. Run lokale tests / open index.html en doe een smoke test.
5. Commit met de voorgestelde message.

Ticketvolgorde staat in `00-roadmap.md`.
